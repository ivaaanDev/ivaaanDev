## 💻 Proyectos Principales

### KokosPOS
*Sistema de Punto de Venta (POS) en evolución para el sector restaurantero (Pizzerías).*
- **Desarrollo:** Implementación de controladores backend fuertemente tipados utilizando **TypeScript**.
- **Hardware:** Integración directa para el manejo de impresión térmica profesional de comandas.
- **Infraestructura:** Refactorización y migración activa de la base de datos local hacia **Supabase** para garantizar alta disponibilidad y escalabilidad.

### Spotify Data Analysis
*Procesamiento y análisis de datos de la industria musical.*
- **Data Engineering:** Importación y limpieza de grandes volúmenes de datos estructurados (CSV).
- **Consultas:** Diseño y ejecución de consultas relacionales avanzadas en **SQL** para la extracción eficiente de métricas.
